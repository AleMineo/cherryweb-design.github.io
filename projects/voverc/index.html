UX/UI design - HTML/SCSS - GITVoverc - Cloud Telephone

Voverc — later acquired and rebranded as Voxloud — was a cloud-based telephony platform offering virtual PBX solutions for businesses. The project involved the design of the corporate website and the UI design and front-end development of the cloud phone system administration panel.

Role

UX/UI Designer, Graphic Designer

Client

Voverc

Tools

Photoshop InVision GIT HTML/CSS VS Code

01 — Project Vision & Strategy

The objective was to position Voverc as a modern, reliable cloud telephony provider while ensuring that its digital ecosystem reflected simplicity and technological efficiency.
Particular attention was given to translating complex telecommunication functionalities into an accessible and intuitive user experience.

Key goals included:

  • Design a clear and conversion-oriented corporate website
  • Create an intuitive administration panel for managing cloud PBX features
  • Ensure visual and functional consistency across marketing and product interfaces

02 — UX & Interface Design

I designed the information architecture and UI components for both the website and the cloud PBX administration panel.
The admin interface required a strong focus on usability, given the technical nature of telephony configuration, call routing, user management, and system settings.

Focus areas included:

  • Simplifying complex configuration flows
  • Designing structured dashboards and control panels
  • Creating reusable UI components for scalability

03 — Design & Front-End Development

Alongside React developers, I contributed to the HTML/CSS implementation of the cloud PBX administration panel, ensuring that the design system was correctly translated into functional UI components.
The corporate website was designed to communicate innovation and clarity, reinforcing the company’s technological positioning.

Key outcomes included:

  • Corporate website UI design
  • Admin panel UI design for cloud telephony management
  • HTML/CSS front-end development in collaboration with React developers
  • Responsive layouts and interface optimization

04 — My Role

I was responsible for:

  • Corporate website design
  • UX and UI design of the cloud PBX administration panel
  • HTML/CSS development in collaboration with the React development team
  • Ensuring visual consistency between marketing and product environments

My contribution helped translate a technically complex telephony product into a structured, intuitive, and visually coherent digital experience.
