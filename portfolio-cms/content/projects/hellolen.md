<div class="description">
  <h4>01 - Project Vision & Strategy</h4>
  <p>Designed and built a visual-first portfolio for a cosplay and modeling artist - balancing immersive imagery with clean navigation across mobile, tablet, and desktop.</p>
</div>

<div class="description">
  <h4>02 - Design & UX Approach</h4>
  <p>Worked without wireframes, following an instinctive, visually driven process. Layout decisions focused on:</p>
  <ul>
    <li>Image-centered compositions that maximize visual storytelling</li>
    <li>Minimal UI elements to keep focus on the artwork</li>
    <li>Consistent aesthetic rooted in anime, manga, and cosplay culture</li>
  </ul>
</div>

<div class="description">
  <h4>03 - Frontend Development & Interaction</h4>
  <p>Manually coded in HTML and SCSS with jQuery for interactions, including:</p>
  <ul>
    <li>Fully responsive layouts across all screen sizes</li>
    <li>Custom gallery structures and dynamic image handling</li>
    <li>Smooth transitions and animations without compromising performance</li>
  </ul>
</div>

<div class="description">
  <h4>04 - My Role</h4>

    <li>Manual frontend development in HTML & SCSS</li>
